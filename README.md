# Loan Acceptance Prediction

## Problem
Predict whether a customer will accept a personal loan.

## Tools
- Python
- Scikit-learn
- Pandas
- Google Colab

## Steps
- Data preprocessing
- Feature engineering
- Model training (Logistic Regression + Decision Tree)
- Evaluation

## Results
- Accuracy: ~90%
